# WeTube

Cloning Youtube with Vanilla and NodeJS

https://murmuring-wildwood-24907.herokuapp.com

## Tech/framework used
---------
- Language: Javascript
- Front-end: HTML5, SCSS, Pug, Webpack
- Back-end: NodeJS, MongoDB, Express
- Deployment: AWS S3, Heroku, Mongo Atlas
